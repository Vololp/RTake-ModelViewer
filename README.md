R-Take ModelViewer is a simple app where you can view models.

This app only supports | gltf, fbx |

If you want to exporting or editing the project make sure you have Godot 4.6 or 4.5. This Application is open sourced plus with frequent update's but not too much. Also This Application is function based, so the functions are in separate scripts to make it easy and organized

And the biggest feature of all ( No AI )

Infinite Ground Grid/xyz | at GodotShader

Liquid Glass Frutiger Aero UI/sentinelcnd | at GodotShader
